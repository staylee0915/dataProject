## 가설검정

모집단과 표본

귀무가설과 대립가설

- 귀무가설 : 기존에 알려진 사실, 효과나 차이가 없음
- 대립가설 : 연구자가 이증하려는 사실, 효과나 차이가 있음.

검정결과 : 검정통계량 : 귀무가설과 데이터 간의 차이를 나타내는 통계적 값

p-value : 유의수준 (0.05)

- 유의수준보다 작으면 귀무가설을 기각하고, 대립가설을 채택한다
- 유의수준보다 크면 귀무가설을 기각하지 못한다 (귀무가설 채택)

가설검정 프로세스
- 통계적 가설 설정 : 귀무가설과 댋가설
- 유의수준 결정
- 검정통계량 및 p-value 유의확률 계산

### t-test

단일표본 검정 (모집단 1개) >> 과자의 무게는 200g 과 다른지 검증
대응표본검정 (모집단 2개) // 같은 집단 >> 신약의 효과 (전후) 검정
독립표본 검정 (모집단 2개) // 서로 다른 집단 >> 1반과 2반의 성적차이 검정


단일표본 검정 
 - 정규성검정
  - 정규성을 따를 경우 : 단일표본 검정 ttest_1samp(data, 기대값)
  - 정규성을 따르지 않을 경우 : 윌콕슨의 부호 순위 검정 wilcoxon(data-기대값)
  
대응표본 검정
 - 정규성검정
   - 정규성을 따를 경우 : 대응표본검정 ttest_rel(data1, data2)
   - 정규성을 따르지 않을 경우 윌콕슨의 부호 순위 검정 wilcoxon(data1, data2)

독립표본 검정
 - 독립표본검정
   - 정규성 검정 
     - 정규성을 따를 경우 : 등분산 검정 levene(data1, data2)
       - 등분산 검정을 따를 경우 독립 표본 검정 ttest_ind(data1, data2)
       - 등분산 검정을 따르지 않을 경우 : 독립 표본 검정 (웰치의 t 검정) // ttest_ind(data1, data2, equal_var = False)
     - 정규성을 따르지 않을 경우
       - 맨 휘트니 U검정 : mannwhitneyu(data1, data2)
